# My Portfolio Website

A modern and responsive personal portfolio website built with React.js to showcase my projects, technical skills, certifications, and contact information.

## 🚀 Features

* Responsive design for desktop, tablet, and mobile devices
* Professional hero section with introduction
* Skills showcase with technology icons
* Project portfolio with live demo and GitHub links
* Project details modal for better user experience
* Contact section for professional inquiries
* Smooth and interactive user interface
* Modern styling using Tailwind CSS and custom CSS

## 🛠️ Technologies Used

### Frontend

* React.js
* JavaScript (ES6+)
* HTML5
* CSS3
* Tailwind CSS

### Libraries

* React Icons
* EmailJS
* React Typed

## 📂 Project Structure

```bash
src/
│
├── assets/          # Images and static resources
├── components/      # Reusable React components
├── App.jsx          # Main application component
├── main.jsx         # Application entry point
└── index.css        # Global styles
```

## 💻 Installation

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project folder:

```bash
cd my-portfolio
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
